# Sorting and Searching

This category includes problems involving:
- Different sorting algorithms
- Binary search variations
- Finding kth elements
- Custom sorting criteria

## Problems in this folder:
- Sort an Array
- Kth Smallest element
- Sort specific patterns (0s, 1s, 2s)

## Key Patterns:
- QuickSort, MergeSort, HeapSort algorithms
- Binary search for O(log n) searching
- Partitioning techniques
- Custom comparators for complex sorting
