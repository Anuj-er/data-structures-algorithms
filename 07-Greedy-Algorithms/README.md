# Greedy Algorithms

This category includes problems that use greedy approach:
- Making locally optimal choices
- Stock trading problems
- Scheduling and optimization
- Discount and pricing problems

## Problems in this folder:
- Best Time to Buy and Sell Stock
- Final Prices With Special Discount

## Key Patterns:
- Choose the best option at each step
- No need to reconsider previous choices
- Often involves finding optimal timing
- Single pass through data when possible
